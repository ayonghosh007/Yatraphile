## 📖 Overview

Yatraphile is a tourism management system developed to simplify travel planning and destination exploration. The platform provides users with an interactive environment where they can browse destinations, build customized travel itineraries, search efficiently using advanced filters, and access valuable insights through user reviews and ratings.

The project focuses on delivering a seamless experience across all devices while maintaining an attractive and user-friendly interface.

## ⚙️ Installation & Setup

1. Clone the repository

```bash

```

2. Navigate to the project directory


